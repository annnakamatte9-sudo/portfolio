Featured Project

Recess Work

Group Project · Recess 2026
UG Scholar
An inclusive digital learning platform built for Uganda and beyond.

UG Scholar is an educational platform designed to make digital learning more accessible across Uganda. Similar in concept to Google Classroom and Moodle, it provides a structured online learning environment — but goes further by addressing two real barriers students face: language diversity and age-appropriate content delivery.

🌍
Language Translation
Supports multiple local languages, breaking down literacy barriers so learners can engage in their native tongue.
🔒
Age Restriction System
Content is filtered by age group, ensuring learners only access material appropriate to their developmental stage.
📚
Course Management
Educators can upload materials, set assignments, and track student progress in a structured digital classroom.
👥
User Roles
Distinct roles for students, educators, and administrators allow for organised, role-based platform interactions.
✦ Unique: Multilingual + Age-Safe Learning
Web Platform User Authentication Database Design Language API Role-Based Access
